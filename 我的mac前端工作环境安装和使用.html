我的Mac前端工作环境安装和使用

发表于 2016-02-14 | 分类于 Npm ， Nodejs ， brew ， git

安装brew

1、可以查看brew官方网站- http://brew.sh/index_zh-cn.html
2、直接安装：打开终端窗口, 粘贴 /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"回车。

Node.js,git安装

打开终端窗口输入brew install node git安装成功。

Npm简介

npm是nodejs的包管理器，全称Node Packaged Modules，通过npm可以下载安装前端包。

我的常用npm方法

1、输入 npm -v 测试是否安装成功
2、Npm安装语法 npm install <Module Name>
3、Npm更新模块 npm update <package>
4、创建简单项目：初始化 npm init之后可以进行需求安装。

我的常用git方法

1、输入git init初始化
2、输入git add -A 把当前文件夹的所有文件添加到git上
3、输入git commit -am “描述信息” 提交到本地的版本库
4、输入git push -u origin master 提交到线上
5、输入git pull 拉取更新线上代码
6、输入git push 提交本地新增代码
7、输入git reset 恢复到上一个版本
8、输入git fetch --all 从服务器拉代码不合并
9、输入git reset --hart origin/master 把本地代码改成刚拉取的代码
10、输入git clone 项目地址克隆git开源项目
git新增ssh key
1、输入ssh-keygen -t rsa -C "git用户名" 生产key
分支
1、输入git checkout -b 分支名 （ 新建分支并进入分支）
2、输入git branch （列出所有分支）
测试
1、输入ssh -T git@github.com 测试github是否可用
2、输入git@git.oschina.net测试oschina是否可用
