Vue相关命令

发表于 2017-01-08 | 分类于 vue

npm install -g vue-cli
-g (全局的) vue－cli（新建vue项目的命令）
vue list (显示所有可用的模板)
vue init list模板 项目文件夹名 （创建项目）
cd 项目文件夹名 （进入文件夹）
npm install （自动根据当前文件夹的package.json安装依赖包或扩展库）
npm run dev （编译，打开浏览器，自动打开页面，页面自动重载）
npm install vue-resource —-save (安装一个新的包并写入package.json)
npm install vue-resourse —-save-dev (安装包并保存到开发依赖)
npm install —production (生产模式安装，只安装dependencies里面的包)
