正则表达式自学文档

发表于 2016-12-13 | 分类于 正则表达式

用法

1、''.match(/正则/i) - i代表忽略大小写
2、''.replace(/正则/g) - g代表全局匹配

正则写法

1、起始符 ^，结束符 $
2、特殊符 ^ $ . * ( ) [ ] { } ? - 需要转义符 \
3、特定范围 [ ]里面的表达式代表一位，例如 [1-5] [a-z]
4、预设范围 \d 写全是digital代表0-9、\w 写全是word代表字母数字下划线、\s 写全是space代表空格，如果写成大写字母代表非它
5、次数 用{}格式
a、固定次数：例如 {5} 代表5次
b、范围次数：例如 {3-5} 代表3-5次
c、几次以上：例如 {4, } 代表4次以上，如果是一次以上可以用 +，0次以上或者任意次数可以用 *
d、几次一下：例如 {0,5} 代表5次以下

关于Sublime3开发工具小知识

发表于 2016-01-08 | 分类于 sublime3

sublime3 注册码

sublime3 安装插件

在sublime3软件里按下ctrl + shift + p快捷键，输入install package 回车，打开插件包管理器，安装自己需要的插件即可。

sublime3 安装Vue插件

按下ctrl + shift + p快捷键，输入Vue即可看到前3个Vue相关的插件，回车安装即可。

sublime3 安装主题和选择主题

按下ctrl + shift + p快捷键，搜索material theme安装主题，安装完成后，按下ctrl + shift + p快捷键，输入material theme 选择active theme启用主题，按上下选择喜欢的主题
